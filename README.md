# Customer Churn Analysis

## Project Overview
This project focuses on analyzing customer data to identify churn patterns and understand factors affecting customer retention. Using SQL queries, business insights were generated to support decision-making and improve customer engagement strategies.

## Objectives
- Analyze customer behavior and churn trends
- Identify factors affecting customer retention
- Generate business insights using SQL
- Support data-driven decision making

## Features
- Customer segmentation
- Churn trend analysis
- Customer retention analysis
- Business KPI reporting
- Data exploration using SQL

## Tech Stack
- MySQL
- SQL

## Dataset
Customer dataset containing:
- Customer ID
- Customer demographics
- Purchase history
- Subscription details
- Churn status

## Analysis Performed
- Total customers analysis
- Churn percentage calculation
- Retained vs churned customers
- Customer category analysis
- Trend identification

## Business Insights
- Identified customer groups with higher churn rates
- Generated insights for improving customer retention
- Supported business decision-making through data analysis
